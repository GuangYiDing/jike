本项目为仿即刻App后端项目，实现动态发布，信息更改，点赞关注，评论回复，广场推荐，消息通知及小程序登录等功能

前端项目请跳转[jike-vue](https://github.com/GuangYiDing/vue-jike)

项目开发文档详见 preview 下开发文档
